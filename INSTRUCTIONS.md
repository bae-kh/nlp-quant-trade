1.사용 기술: Python 3.10 이상을 사용하며, 거래소 API 연동은 반드시 ccxt 라이브러리를 사용한다.

2.보안: API 키(Access Key, Secret Key)는 절대 코드 내부에 텍스트로 작성하지 않는다. 반드시 os.environ을 통해 .env 파일에서만 불러온다.

3.폴더 접근 권한 제한:

    core/ 폴더 내의 파일(계좌 연동 로직 등)은 절대 수정하거나 삭제하지 않는다. (읽기 전용)

    새로운 매매 전략 코드는 반드시 strategies/ 폴더 내에 새로운 파일로만 생성한다.

4.안전장치: 모든 거래소 API 호출에는 반드시 try-except 구문을 적용하여, 네트워크 에러나 타임아웃 발생 시 프로그램이 비정상 종료되지 않고 로그를 남기도록 처리한다.
